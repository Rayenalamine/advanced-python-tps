\# TPs Avancés Python



Ce dépôt contient les travaux pratiques (TPs) du cours \*\*Advanced Python\*\* de Teknolabs.  

Chaque TP est dans son propre dossier.



\## Structure du dépôt



1\. \*\*fastapi-postgres/\*\*  

&#x20;  - TP sur \*\*FastAPI avec PostgreSQL\*\*  

&#x20;  - Contient un API avec \*\*CRUD\*\* pour la table `users`  

&#x20;  - Point d’accès Swagger UI : `http://127.0.0.1:8000/docs`  



2\. \*\*web-scraping/\*\*  

&#x20;  - TP sur le \*\*Web Scraping\*\*  

&#x20;  - `scraper.py` ou `simple\_scraper.py` : récupère le titre, les headers et les liens d’une page web  



3\. \*\*langchain/\*\*  

&#x20;  - TP sur \*\*LangChain\*\* (dossier vide avec `.gitkeep`)  



4\. \*\*crawl4ai/\*\*  

&#x20;  - TP sur \*\*Crawl4AI\*\* (dossier vide avec `.gitkeep`)  



\---



Tous les dossiers sont prêts pour être testés ou exécutés.  

Les fichiers `.gitkeep` sont utilisés pour suivre les dossiers vides avec Git.
